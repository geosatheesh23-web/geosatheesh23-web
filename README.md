# 🎮 GameHub

🔗 **Live Demo:** https://geosatheesh23-web.github.io/geosatheesh23-web/index.html

A fast, clean, no-nonsense web game hub built using pure HTML, CSS, and JavaScript.

---

## 🚀 Overview

GameHub is a lightweight browser-based gaming site focused on speed, simplicity, and smooth user experience. No frameworks, no heavy assets — just instant play.

---

## 🎯 Features

* ⚡ Fast loading (static site)
* 🎮 Multiple mini games
* 📱 Fully responsive design
* 🌙 Light / Dark mode toggle
* 👤 Local user system (guest/login)
* 🧭 Clean and simple UI

---

## 🕹️ Games

* **2048** – Sliding number puzzle
* **Tic Tac Toe** – Classic strategy game
* **Reaction Test** – Measure your reflex speed
* **Memory Match** – Card matching game
* **Rock Paper Scissors** – Play vs computer

---

## 🧠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
* LocalStorage (for user + settings)
